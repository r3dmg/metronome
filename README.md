[# Метроном Веб-Приложение]([url](https://r3dmg.github.io/metronome/))

## Структура проекта

- **frontend/**: Клиентская часть приложения
  - `index.html` — Основная HTML-страница
  - `styles.css` — Стили приложения
  - `script.js` — Логика приложения на JavaScript

- **backend/**: Серверная часть (пока пустая, так как приложение полностью клиентское)

## Функционал

- Метроном с настройками BPM, размера такта, ударных
- Автоматическое увеличение BPM
- 5 профилей настроек, переключаемых по табам
- Отсчет 3 секунды перед стартом с звуковым сопровождением (опционально)
- Общее время работы

## Запуск

Запустите локальный сервер в папке frontend:

```bash
cd frontend
python3 -m http.server 8000
```

Затем откройте http://localhost:8000 в браузере.
